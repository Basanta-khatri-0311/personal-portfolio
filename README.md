# Basanta Khatri – Portfolio 🚀

Welcome to my personal portfolio repository! This is a high-performance, premium web experience crafted with **React**, **Vite**, and **Tailwind CSS**. It serves as a digital showcase of my technical skills, creative design aesthetic, and professional journey as a Frontend Developer.

---

## ✨ Key Features

- 💎 **Premium "True Black" Aesthetic:** A sophisticated dark-themed UI (#08080f) with subtle ambient blue accents and glassmorphism.
- 📦 **Horizontal Work Carousel:** A high-impact, single-item project showcase with smooth snappable horizontal scrolling and custom navigation.
- 🔗 **Clean URL Navigation:** Custom smooth-scrolling logic that eliminates the `#` hash from the browser address bar for a seamless app-like feel.
- 📱 **Fully Responsive Design:** Optimized for every device, from high-resolution desktop monitors to compact mobile screens.
- 📮 **Interactive Contact Form:** Fully functional email submission system integrated with **EmailJS**.
- ⚡ **Vite Optimized:** Extremely fast load times and optimized asset delivery for a smooth user experience.

---

## 🛠️ Technology Stack

- **Framework:** [React](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Remix Icon](https://remixicon.com/) & [Lucide](https://lucide.dev/)
- **Form Handling:** [EmailJS](https://www.emailjs.com/)
- **Animations:** Custom CSS Reveal Animations & Framer-like transitions.

## 🚀 Getting Started

To run this portfolio project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Basanta-khatri-0311/personal-portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file for your EmailJS service, template, and public keys (if applicable).

4. **Run the development server:**
   ```bash
   npm run dev
   ```
